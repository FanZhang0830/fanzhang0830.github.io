My work focuses on connecting satellite data, open geospatial APIs, watershed models, and interactive WebGIS interfaces to support better environmental monitoring, planning, and risk assessment.

Featured Project

EcoHydroAI Watershed Intelligence Platform

EcoHydroAI is an open-source WebGIS prototype that integrates watershed boundaries, live weather forecasts, remote sensing layers, and environmental decision-support indicators into an interactive map-based platform.

  • Interactive watershed map built with Python, Folium, and open geospatial data
  • FastAPI backend deployed on Render for live weather forecast requests
  • GitHub Pages frontend for lightweight public portfolio deployment
  • Designed for future Sentinel-2 land cover statistics, Raven/HMS model outputs, and AI-generated summaries
